What's New in The Costumemaster

The latest feature update of The Costumemaster, v2.2.0, includes great new features and improvements.

  • The game now supports localization.
  • Adds an option to turn off physics-based player movement in Settings › General.

*Excludes debugging options in Preferences › Advanced.

Previous feature releases

v2.1.0

  • Settings have been overhauled and feel more at home with macOS.
    • Adds support for changing the app's icon when running with new custom icons.
    • Adds support for changing Game Center leaderboards and achievements settings in Preferences › Game Center.
    • Improves setting feedback with instantaneous responses*.
  • Improves the user interface for the AI Simulator Tool.
  • Adds the Teal Converse agent as an agent type to the AI Simulator.
  • Updates Game menu by making Game › Open Level... always visible.
  • Resolves some issues in levels from the Watch Your Step! DLC.
  • Adds support for creating state assessment recordings for machine learning agents.
  • Introduces a brand-new soundtrack unique to The Costumemaster.

v2.0.0

  • Adds the AI Simulator Tool to run AI simulations with different agents and levels.
  • Changes the target radius for exit doors to make it easier to go to the next level.
  • Adds improvements to starting a new game, resuming games, and loading a given level using the menu bar.
  • Adds support for submitting scores to daily challenge leaderboards.
  • Adds the Game Center access point on Macs running macOS 11 Big Sur.
  • Adds a new Help Book to the app.
